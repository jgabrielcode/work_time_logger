# work_time_logger
Sistema de Registro de Ponto

📌 Visão Geral

Este projeto foi criado para resolver um problema prático enfrentado por um salão de costura, que atualmente realiza registros manuais de ponto. Esse método gera falta de controle, inconsistências e dificuldades na gestão da jornada de trabalho. O sistema digital de registro de ponto busca oferecer uma solução simples, confiável e acessível.

🎯 Funcionalidades

Registro de entrada e saída dos funcionários

Integração com Google Sheets

Interface amigável, pensada para pequenos negócios

🚀 Objetivos

Substituir registros manuais por um sistema digital confiável

Reduzir erros e aumentar a transparência no controle de jornada

Facilitar a gestão de equipes em pequenos negócios

Criar uma solução escalável que pode ser adaptada para outros setores

🛠️ Tecnologias

Frontend: HTML , CSS & JAVA SCRIPT

Banco de Dados: Google Sheets
